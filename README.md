# RackOnWheels

Simple framework to build web applications base on rack.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rack_on_wheels'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rack_on_wheels

## Usage

Start example app:

    ruby app/main.rb

Go to http://localhost:8080/ to see the page

## Development

Run unit tests:

    bundle exec rake
